# open-ports
Shows open TCP and UDP ports on your machine.

## Tested Platforms
- Pardus 23.4
- Linux Mint 22.2

## Requirements
Python 3, psutil, colorama

## Running
Just pass `python3 ports.py` on the project root directory's command line.
